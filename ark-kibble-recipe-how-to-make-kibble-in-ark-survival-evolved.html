ARK Kibble Recipe: How to Make Kibble in ARK Survival Evolved

Publish date: 2024-08-24

If you’re looking for the best way possible to start taming creatures in ARK: Survival Evolved, you’re going to need to start making kibble as soon as you possibly can. You’ll want to use this to start adding new creatures to your civilization. You’ll be able to tame them, ride them, feed them, and make them your allies in battle, so they are an invaluable tool to have in your arsenal.

Let’s find out what you’ll need to start taming creatures, and what materials you’ll need to start making kibble to keep them happy, and bring them home with you.

How To Make Kibble In ARK: Survival Evolved

The first thing you’ll need to make sure that you have available for use is the Cooking Pot, which allows you to not only craft food for yourself, but for your new friends that you’ll find in the wild. After you have made sure that your pot is available for use, you’ll need to find and gather some materials in the wild to make sure it can run, including:

  • Thatch – Burns for 7.5 Seconds
  • Wood – Burns for 30 Seconds
  • Sparkpowder – Burns for 1 Minute
  • AnglerGel – Burns for 4 Minutes

If you do not have a cooking pot, or if you are looking to create another one to make sure you can continue to make all different types of kibble, you’ll need:

  • 75 Stone
  • 15 Thatch
  • 10 Wood or Fungal Wood
  • 5 Flint

After you have created your cooking pot or added another to your encampment, you’ll want to start creating kibble as soon as possible to start bringing new creatures your way. Follow along with our table here to learn what you’ll need to start making kibble as soon as possible.

Also, keep an eye peeled out for all of the different patch notes that are available so you can make sure that you’re always up to date on the new features being added to the game!
